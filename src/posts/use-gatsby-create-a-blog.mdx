## setp1 安裝 Gatsby

依據[Gatsby 官網](https://www.gatsbyjs.org/)的說明，利用 npm 於全域安裝 gatsby-cli

```
install -g gatsby-cli
```

## setp2 建立專案

```
gatsby new [SITE_DIRECTORY_NAME] [URL_OF_STARTER_GITHUB_REPO]
```
